# Annual Summary Feature

## 目的

一年単位で収入、固定費、変動費、収支をざっくり確認できるようにする。

---

## 画面

- パス: `/annual-summary`
- 画面名: 年間サマリー画面

---

## 表示内容

- 年間全収入
- 年間固定費
- 年間変動費
- 年間実収支
- 月別出費推移
- 年間収入の使い道円グラフ（固定費、変動費、年間実収支の排他的な内訳）
- 年間貯蓄率
- 固定費率
- 最も変動費が多かった月
- 前月比

---

## 実装順序

年間サマリーはMVP対象とする。

ただし、実装順序としてはトップ画面、収入画面、固定費画面、カレンダー画面の後に着手する。

---

## 計算方針

```txt
年間全収入
= その年に登録された全収入

年間使える収入
= included_in_balance = true の収入合計

年間貯める収入
= included_in_balance = false の収入合計

年間固定費
= 各月の対象固定費合計の年間合計

年間変動費
= その年に登録された固定費以外の出費合計

年間実収支
= 年間全収入 - 年間固定費 - 年間変動費

年間収入の使い道
= 年間固定費、年間変動費、年間実収支を排他的な内訳として表示する

年間生活費残り
= 年間使える収入 - 年間固定費 - 年間変動費

年間貯蓄率
= 年間実収支 / 年間全収入 * 100

固定費率
= 年間固定費 / 年間全収入 * 100

前月比
= 実収支がある最新月の実収支 - その前月の実収支
```
