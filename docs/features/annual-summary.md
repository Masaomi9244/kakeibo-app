# Annual Summary Feature

## 目的

一年単位で収入、固定費、出費、収支をざっくり確認できるようにする。

---

## 画面

- パス: `/annual-summary`
- 画面名: 年間サマリー画面

---

## 表示内容

- 年間全収入
- 年間使える収入
- 年間固定費
- 年間出費
- 年間実収支
- 月別出費推移
- 年間収支内訳円グラフ
- 最も支出が多かった月

---

## 実装順序

年間サマリーはMVP対象とする。

ただし、実装順序としてはトップ画面、収入画面、固定費画面、カレンダー画面の後に着手する。

---

## 計算方針

```txt
年間全収入
= その年に登録された全収入

年間使える収入
= included_in_balance = true の収入合計

年間貯める収入
= included_in_balance = false の収入合計

年間固定費
= 各月の対象固定費合計の年間合計

年間出費
= その年に登録された出費合計

年間実収支
= 年間全収入 - 年間固定費 - 年間出費

年間生活費残り
= 年間使える収入 - 年間固定費 - 年間出費
```
