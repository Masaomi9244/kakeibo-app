# Incomes Feature

## 目的

収入を登録・編集・削除し、今月使える金額へ含めるかを管理できるようにする。

---

## 画面

- パス: `/incomes`
- 画面名: 収入画面

---

## 表示・操作

- 収入を複数登録できる
- 収入を編集できる
- 収入を削除できる
- 収入ごとに「今月使えるお金に含める」を設定できる

---

## 入力項目

- 金額
- 収入日
- メモ
- 今月使えるお金に含めるか

---

## `included_in_balance`

ボーナスや貯金したい臨時収入は、収入として記録しつつ、今月残り使える金額の計算から除外できる。

```txt
included_in_balance = true
  今月使える金額に含める

included_in_balance = false
  収入として記録するが、今月使える金額には含めない
```

年間実収支には、`included_in_balance` の値に関係なく全収入を含める。
